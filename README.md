# 该版本支持将程序添加至系统环境变量

# 麦麦启动器v3.4.2PATH支持版

运行安装包以安装程序，安装完成后会**自动将安装目录添加至系统环境变量**，除了在开始菜单或桌面上通过快捷方式启动等，支持在任何命令行中通过输入`mbl`的方式启动。
使用详情等请见[`main`](https://github.com/xiaoCZX/MaiMbot-initiate)分支
